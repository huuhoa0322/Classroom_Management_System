Hãy tạo file api_design_rules.md lưu vào thư mục gốc chuẩn bị AI Onboarding với những yêu cầu như sau, sử dụng XML Tags cho mỗi phần:
- Global route prefix: `/api/v1`
- URL Path: sử dụng quy tắc kebab-case
- Response format: wrapped in { "code", "message", "data" }
- Rule: Apply JWT Bearer Authentication to all secured endpoints.
- Never return sensitive fields like 'password' in response schemas.
Example: 
Example:
  /api/v1/auth/login:
    post:
      summary: "User Login"
      security:
        - bearerAuth: [] # AI will learn to append this to all locked routes
      responses:
        '200':
          content:
            application/json:
              schema:
                properties:
                  code: { type: integer, example: 200 }
                  message: { type: string }
                  data:
                    {accessToken, email, fullName, roles[]}

Sử dụng tiếng anh, mô tả đầy đủ thông tin, rõ ràng, dễ hiểu.
